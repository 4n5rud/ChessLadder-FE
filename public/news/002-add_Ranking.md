# ChessLadder 랭킹 페이지와 소식 페이지 추가!

ChessLadder를 이용해 주시는 사용자 여러분께 감사드립니다. 이번 업데이트를 통해 사용자 간의 실력을 확인하고 서비스의 최신 동향을 파악할 수 있는 두 가지 핵심 기능이 추가되었습니다!

--------------------------------

## 1. 소식 페이지 (News Page)
사용자와의 원활한 소통을 위해 통합 공지 시스템인 소식 페이지를 구축하였습니다.

주요 기능
접근 방법: 상단 내비게이션 바의 "소식(News)" 메뉴를 통해 접속 가능합니다.

업데이트 아카이브: 시스템 변경 사항, 새로운 기능 출시, 점검 안내 등 모든 공식 공지사항을 기록합니다.

정렬 방식: 사용자 편의를 위해 최신순 정렬(Latest First) 방식을 적용하여 최상단에서 가장 최신 소식을 바로 확인하실 수 있습니다.

날짜 표기: 각 게시물별로 작성 날짜를 명시하여 업데이트 시점을 명확히 알 수 있도록 구성했습니다.

---------
 
## 2. 랭킹 페이지 (Ranking Page)
가입자들의 객관적인 실력을 지표로 나타내고 경쟁의 즐거움을 제공하기 위한 랭킹 페이지가 도입되었습니다.

기능 목적
ChessLadder 가입자들의 데이터를 기반으로 레이팅(Rating) 및 티어(Tier) 정보를 제공합니다.

전체 사용자 중 자신의 위치를 파악하고 실력 향상의 척도로 활용할 수 있습니다.

상단 메뉴의 "랭킹(Ranking)" 탭에서 확인할 수 있습니다.
데이터 동기화 주기에 따라 실시간 반영에는 약간의 차이가 발생할 수 있습니다.
